# Minecraft Server Builder (MSB) - プロジェクト憲章

## 1. プロジェクトビジョン
Minecraftサーバーの構築・管理を、最も安全、高速、かつ型安全に自動化するRustライブラリを提供する。

## 2. 現在のステータス：Phase 1 実装完了 🚀
- **Tauri GUI統合:** React/TypeScriptによる洗練されたダッシュボードの完成。
- **マルチエンジン対応:** Purpur, Paper, Folia, Arclight, Fabric, Vanilla, BDSの完全サポート。
- **並列オーケストレーション:** Backend, Proxy (Velocity), Limboの非同期並列デプロイと自動起動スクリプト生成の実装完了。

## 4. コーディング規約
- **安全性:** パフォーマンス上不可避な場合を除き、unsafeは禁止。
- **非同期:** tokioをランタイムに、reqwestをHTTPクライアントに使用。
- **型定義:** 全ての外部データ構造はserdeで型定義する。
- **エラー処理:** ライブラリ内部はthiserror、応用アプリはanyhowを使用。
- **ドキュメンテーション:** すべてのパブリック関数および複雑なロジックには、指定のJSDoc/Rustdoc形式で必ずドキュメントコメントを付与する。
